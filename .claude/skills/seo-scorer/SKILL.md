# SeoScorer

## 역할
수정된 초안을 SEO/GEO/AEO 3영역으로 채점하고, 85점 미만 항목을 개선하여 반영한다.

---

## 채점 기준 (총 120점 만점)

### SEO 영역 (70점) — 가중치 40%

| 항목 | 기준 | 배점 |
|------|------|------|
| Title tag | Primary 키워드 포함, 50-60자 | 10 |
| Meta description | 키워드 포함, 150-160자, CTA 포함 | 8 |
| URL slug | 짧고 키워드 포함, kebab-case | 5 |
| H1 | Primary 키워드 포함, 검색 의도 매칭, 1개만 | 10 |
| H2 구조 | Secondary 키워드 활용, 토픽 커버리지 | 10 |
| 첫 100자 | Primary 키워드 자연스럽게 포함 | 5 |
| 내부 링크 | 3-5개 관련 페이지 링크 | 7 |
| 외부 링크 | 2-3개 권위있는 출처 | 5 |
| 키워드 밀도 | 1-2% (과도하지 않게) | 5 |
| Schema markup | FAQ, HowTo, Article 중 해당 적용 | 5 |

### GEO 영역 (30점) — 가중치 35%

| 항목 | 기준 | 배점 |
|------|------|------|
| Q&A 구조 | FAQ 섹션 포함, PAA 질문 답변 커버 | 10 |
| 인용 가능한 정의문 | 명확한 팩트 기반 문장, AI가 인용할 수 있는 형태 | 10 |
| 데이터/통계 포함 | 구체적 수치, 출처 명시 | 5 |
| 구조화된 데이터 | 표, 리스트 등 AI가 파싱하기 좋은 구조 | 5 |

### AEO 영역 (20점) — 가중치 25%

| 항목 | 기준 | 배점 |
|------|------|------|
| 직접 답변 제공 | 질문에 대한 명확한 답변이 본문 초반에 | 10 |
| 단계별 설명 | How-to 형식의 단계별 가이드 | 5 |
| 비교/대조 표 | 선택지 비교가 필요한 경우 표로 정리 | 5 |

---

## 가중 평균 계산

```
가중 평균 = (SEO 득점/70 × 40) + (GEO 득점/30 × 35) + (AEO 득점/20 × 25)
```

**85점 이상 필수.** 미달 시 해당 영역 개선 후 재채점.

---

## 출력물
1. **채점표** (항목별 득점 + 가중 평균)
2. **개선 방향** (85점 미만 항목별 구체적 수정 지침)
3. **개선 반영된 초안** → QualityChecker에 전달
