# 검색 의도 4분류 가이드

| 의도 | 사용자가 원하는 것 | 적합한 콘텐츠 포맷 | 예시 쿼리 | 매칭 CJ Stage |
|------|------------------|------------------|----------|-------------|
| Informational (정보형) | 무언가를 배우고 싶다 | 가이드, 튜토리얼, 설명글 | "백엔드 개발자 되려면" | CJ-1, CJ-3, CJ-5 |
| Commercial (상업형) | 구매/선택 전 비교 | 비교글, 리스트, 리뷰 | "코딩 부트캠프 비교 2025" | CJ-1, CJ-4 |
| Transactional (거래형) | 바로 행동하고 싶다 | 서비스 페이지, 가격 페이지 | "잡코리아 이력서 작성" | CJ-2 |
| Navigational (탐색형) | 특정 페이지를 찾는다 | — (타겟하지 않음) | "잡코리아 로그인" | — |

## 의도 판별 방법
1. Google에서 키워드 검색
2. 상위 10개 결과의 콘텐츠 포맷 확인
3. 리스트가 대다수 → Commercial, 가이드가 대다수 → Informational
4. **SERP 포맷을 거스르면 진다** — 상위 결과가 리스트면 리스트를 쓴다

## CJ Stage별 주요 의도
- CJ-1 (이직/직장): Informational + Commercial 혼합
- CJ-2 (이력서/자소서): Transactional + Informational
- CJ-3 (인터뷰): Informational 중심
- CJ-4 (연봉/오퍼): Commercial + Informational
- CJ-5 (커리어 개발): Informational 중심
- CJ-6 (뉴스/트렌드): Informational
