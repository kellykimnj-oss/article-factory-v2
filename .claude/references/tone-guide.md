# 톤앤매너 가이드

## 기본 톤

| 항목 | 기준 |
|------|------|
| 말투 | 해요체 (존댓말 + 친근한 톤). "~합니다"체 금지 |
| 호칭 | "여러분", "당신"보다는 직접 말하듯 자연스럽게 |
| 문장 길이 | 한 문장 40자 이내 권장. 모바일 가독성 우선 |
| 단락 길이 | 3~4문장 이내. 긴 단락은 소제목으로 분리 |
| 톤 키워드 | 신뢰감 + 전문성 + 친근함. 강의하듯 X, 동료와 대화하듯 O |

## 권장 / 금지 표현

### AI투 금지 표현
```
~할 수 있습니다 / 매우 중요합니다 / 아래와 같습니다 / 다음과 같습니다 /
다양한 방법 / 다양한 측면 / 여러 가지 / 먼저·다음으로·마지막으로(나열) /
중요한 것은~ / 핵심은~ / 이를 통해 / ~라고 할 수 있어요 / ~에 대해 알아보겠습니다
```

### 권장 대체 표현
| 금지 | 대체 |
|------|------|
| ~할 수 있습니다 | ~하면 돼요, ~해보세요 |
| 매우 중요합니다 | (구체적 이유를 직접 서술) |
| 아래와 같습니다 | (바로 내용 시작) |
| 다양한 방법이 있습니다 | (구체적 방법을 바로 나열) |
| 먼저~, 다음으로~, 마지막으로~ | (자연스러운 흐름으로 전환) |
| ~에 대해 알아보겠습니다 | (주제를 바로 시작) |

## 글 구조별 톤

| 구성 요소 | 톤앤매너 |
|----------|---------|
| 제목 | 검색 키워드 포함 + 구체적 숫자/혜택. 자극적 X, 실용적 O |
| 도입부 | 독자의 현재 상황/고민 먼저 언급 → 공감 → 이 글이 해결해줄 것 암시 |
| 본문 | 실제 사례/데이터 중심. 추상적 조언 X. "실제로~", "예를 들어~" |
| 정보 박스 | 꿀팁, 주의사항은 별도 박스로 시각적 구분 |
| CTA | 자연스러운 다음 행동 유도. 강매 톤 X |

## HTML 구성 요소

| 요소 | 규칙 |
|------|------|
| 핵심 요약 박스 | 글 최상단. 3~5줄로 핵심 정보 요약 |
| 목차 | 6,000자 이상 아티클 필수. 클릭 이동 가능 |
| 정보 박스 (팁) | 실무 꿀팁, 추가 정보. 본문과 시각적 구분 |
| 주의 박스 | 흔한 실수, 주의사항. 경고 톤이되 위협적이지 않게 |
| 인용 박스 | 전문가 인용, 통계 데이터. 출처 명시 |
| 비교 표 | 선택지 2개 이상일 때. 한눈에 비교 가능 |
| 단계별 가이드 | How-to 형식. 번호 + 각 단계 1~2줄 |
| FAQ 섹션 | 글 하단. 최소 3개. PAA 질문 반영 |
| CTA 섹션 | 글 최하단. 잡코리아 서비스 자연스럽게 연결 |
