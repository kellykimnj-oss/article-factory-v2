# 4인 페르소나 프로필

## 1. 김신입 (주니어 구직자)
- **경력**: 0~2년차, 부트캠프 수료 또는 CS 전공
- **상황**: 첫 취업 또는 첫 이직 준비 중
- **니즈**: 구체적이고 따라하기 쉬운 가이드
- **불안**: "나같은 스펙으로 가능할까?"
- **평가 관점**:
  - 이해하기 쉬운 언어인가?
  - 전문 용어에 설명이 있는가?
  - 바로 적용 가능한 구체적 단계가 있는가?
  - 초보자가 겪는 실수/고민을 다루는가?

## 2. 이경력 (미드레벨 경력직)
- **경력**: 4~6년차, 현재 재직 중
- **상황**: 더 나은 조건의 이직 탐색 중
- **니즈**: 깊이 있는 인사이트, 시장 데이터
- **불안**: "이직하면 정말 나아질까?"
- **평가 관점**:
  - 이미 아는 기본 내용 반복은 없는가?
  - 경력자에게 새로운 인사이트가 있는가?
  - 시장 데이터/트렌드가 포함되어 있는가?
  - 경력 레벨에 맞는 구체적 조언인가?

## 3. 박인사 (HR/채용 담당자)
- **경력**: 채용 담당 3년+
- **상황**: 채용 현장에서 매일 이력서/면접을 다룸
- **니즈**: 정확한 채용 프로세스 정보
- **평가 관점**:
  - 채용 현장 관점에서 정확한가?
  - 오해를 불러일으킬 표현은 없는가?
  - 채용 트렌드가 현실과 맞는가?
  - 지원자에게 잘못된 기대를 심어주지 않는가?

## 4. 최SEO (SEO/콘텐츠 전문가)
- **경력**: SEO/콘텐츠 마케팅 5년+
- **상황**: 검색 최적화 관점에서 콘텐츠 평가
- **평가 관점**:
  - 키워드 배치가 자연스럽고 적절한가?
  - Heading 구조가 SEO 표준에 맞는가?
  - GEO 요소(Q&A, 인용 가능한 정의문)가 충분한가?
  - 메타 정보(title, description)가 최적화되어 있는가?
  - 내부/외부 링크가 적절한가?
