# 경쟁사 차별화 기준

## 경쟁사 약점

| 플랫폼 | 약점 |
|--------|------|
| 퍼블리 | 이론·원칙 위주, 실전 예시 부족 |
| 원티드 | 개인 사례 치중, 일반화 어려움 |
| 캐치 | 수동적 교정, 구조 가이드 없음 |

## 잡코리아 차별화 3원칙

### 1. Before/After 실례 시각화
텍스트 나열 대신 .ba 클래스로 ❌→✅ 직접 비교

### 2. 오늘 바로 쓸 수 있는 실전 도구
원칙·이론 → 체크리스트, 템플릿, 문장 패턴으로 변환

### 3. 수치·근거 밀도
추상 표현 → 2023~2026년 출처 기반 수치
